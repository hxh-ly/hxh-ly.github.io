# User-Agent

  • 包含创建请求的浏览器和用户代理名称传递服务器

  • 如果经过代理服务器，中间可能加上代理服务器名称。

  • 是什么？是一种向访问网站提供你所使用的浏览器类型、操作系统及版本、CPU 类型、浏览器渲染引擎、浏览器语言、浏览器插件等信息的标识。

  • 作用:判断什么方式打开 是app还是浏览器决定你看到的东西不同。

# Referrer

  • 请求的URI是从哪个页面发出去的
  • 作用：防盗链（只有自己的站点能访问自己的图片资源服务器）
  • 空Referer 不是从链接访问。比如从地址栏凭空输入访问的。
